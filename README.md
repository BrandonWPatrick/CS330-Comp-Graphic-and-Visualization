# CS330-Comp-Graphic-and-Visualization

Designing Software

When I design software, I usually start by figuring out the main goal and then breaking it into smaller parts. For this 3D scene, I focused first on where objects should go, how big they should be, and how the lighting and camera would look. After that, I adjusted details to make everything feel balanced. This project helped me get better at thinking about space and how things fit together visually. I mostly worked in small steps and kept changing things until it looked right.

Developing Programs

I normally start with something basic that works and then build on top of it. In this project, I created simple shapes first, then added movement, transformations, and lighting. I had to test things often and fix small issues as I went. Over the milestones, I got more comfortable making changes carefully instead of guessing.

Impact on My Goals

This project improved both my coding and creative thinking. Learning about graphics and visualization gives me skills that can help in future classes and possibly careers involving visual or interactive systems.
